## 3. Required Dispatch Decision Inputs

The dispatch recommendation must consider:

- Feeder criticality.
- Technician certification.
- Lockout-tagout readiness.
- PPE availability.
- Weather and travel risk.
- Customer ETA obligation.
- Backup crew availability.
- Prior SLA breach history.

Nearest crew should not automatically win the recommendation if certification is expired or unknown.

## 4. Customer Communication Requirement

PrithviGrid requires ETA, risk summary, and backup crew details before 3 PM. Hospital feeder incidents require updates every two hours until restoration. The update should contain:

1. Current transformer status.
2. Assigned certified crew.
3. Safety approval status.
4. Estimated arrival and restoration window.
5. Backup plan if rain delays travel.
6. Next update time.
