# 5-Minute Demo Script

## 0:00 - 0:40 Landing

Open `http://localhost:3000`.

Say: "Flow360 is an agentic next best action platform for workforce teams. It is designed as reusable decision infrastructure, not a chatbot."

Point out the cinematic hero: "The landing page uses an original coded scene inspired by premium enterprise AI SaaS pages to position Flow360 as an agentic decision platform."

## 0:40 - 1:20 Operator Console

Open `/app`.

Show:
- Northstar Health account.
- Critical requirements.
- SLA breach risk.
- Renewal exposure.
- Memory confidence.

Say: "The account manager sees the decision workspace immediately. No chat prompt is required."

## 1:20 - 2:20 Run Planner

Click `Run Planner`.

Say: "The planner orchestrates ingestion, retrieval, analysis, recommendation, and memory preparation."

Show Agent Trace as each stage appears after the run.

## 2:20 - 3:20 Recommendations

Open the top recommendation.

Say: "Each next best action includes owner, due date, priority, confidence, rationale, business metric, and evidence."

Highlight:
- Priya N. license verification.
- CFO cost-risk approval brief.
- Executive renewal-risk check-in.

## 3:20 - 4:10 Evidence And Memory

Show evidence drawer and persistent memory.

Say: "This is not naive RAG. We separate raw, semantic, episodic, profile, and rule memory. Accepted and rejected actions become future context."

## 4:10 - 4:45 Human Review

Click `Approve` on the top action.

Say: "Human-in-the-loop review controls execution and writes reviewer feedback into memory."

## 4:45 - 5:00 Close

Say: "The same platform can be reused for staffing, customer success, SaaS sales, energy field service, or any workflow where business users need explainable recommendations."
